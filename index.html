Why is it awesome?

JazzCSS is awesome because it's a lightweight, flexible, and easy-to-use CSS framework. It's designed to be modular, responsive, and customizable, making it a great choice for developers who want to create custom styles without having to start from scratch. The framework is well-documented and provides a set of pre-built classes that you can use to quickly style your HTML elements. Overall, JazzCSS offers a good balance of flexibility and convenience, and it's a great tool for improving productivity and creating cleaner code.



Geting Started

JazzCSS is a lightweight and easy-to-use CSS framework that can be used to build responsive and modern web designs. This section will guide you on how to get started with JazzCSS and start building your web projects.

Installation

There are several ways to install JazzCSS:


Option 1: Download from GitHub

You can download JazzCSS directly from our GitHub repository. Simply click the "Code" button and select "Download ZIP". Once downloaded, extract the files and copy the jazz.css and jazz.min.css files into your project directory.

Option 2: Install via npm

You can also install JazzCSS via npm. Open your terminal and run the following command:


npm install @thunderk3g/jazz-css@1.0.0

This will install JazzCSS into your project's node_modules directory.

Colors

JAZZCSS strongly empasizes on customization. You can set the primary, secondary and tertiary colors that you might need in your application , JazzCSS will create relevant utility class for those . Once the base colors are set custom text , background and hover utility classes can be generated :


"primary": $primary ,
"secondary": $secondary,
"error": $error,
"info": $info,
"blue": #1919e6,
"red": #e61919,
"yellow": #e6e619,
"green": #19e635,
"orange": #ffa600,
"purple": #9900ff,
"gray": #808080,
"black": black,
"white": white,
"transparent": transparent
